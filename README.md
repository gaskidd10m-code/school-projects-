# School Projects

A collection of Dart projects for school assignments.

## Projects

### Quadratic Equation Solver
A Dart console application that solves quadratic equations using the quadratic formula.

**Features:**
- Calculates discriminant
- Finds two distinct real roots
- Handles repeated roots
- Detects complex roots

**Usage:**
```bash
dart run FCM.41.018.038.24.dart
```

Enter the coefficients a, b, and c when prompted.

## Requirements
- Dart SDK

## Installation
1. Clone the repository
2. Run the Dart files directly or use `dart run filename.dart`
